### Architecture Summary

Agent-Kit is a **local-first, multi-agent orchestration framework** distributed as an NPM package. It scaffolds a hidden `.agent-os` directory into any project, turning any AI IDE (Cursor, VS Code, Windsurf) into a governed AI software company.

**It does NOT run its own AI models.** It augments your existing AI IDE by providing structured roles, governance protocols, and domain-specific intelligence. The AI model (GPT, Claude, Gemini) is the hardware; Agent-Kit is the operating system.
