# React Native Mobile Starter

Cross-platform mobile excellence for Agent-Kit projects.

## Tech Stack
- Framework: React Native with Expo
- Navigation: React Navigation / Expo Router
- Styling: NativeWind (Tailwind CSS)

## Mobile Principles
- Touch-first interactive zones.
- 60fps gesture-driven animations.
- Offline-first persistence (AsyncStorage / SQLite).
- Battery-conscious layout updates.
