# FastAPI Backend-Only Starter

Industrial-grade backend boilerplate for Agent-Kit projects.

## Tech Stack
- Runtime: Python 3.12+
- Framework: FastAPI
- Database: SQLite (SQLAlchemy) / PostgreSQL
- Security: Argon2 hashing with JWT

## Patterns
- Dependency Injection (FastAPI Depends)
- Service/Repository pattern
- Mandatory Pydantic models for request/response validation
