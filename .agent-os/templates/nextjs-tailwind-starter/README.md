# Next.js + Tailwind CSS Starter

This is a premium boilerplate for Agent-Kit projects.

## Tech Stack
- Framework: Next.js 15 (App Router)
- CSS: Tailwind CSS 4
- Icons: Lucide / Heroicons
- UI Library: Pure Tailwind (No Default)

## Rules
- Strictly follow the Purple Ban.
- Use topological betrayal in the Hero section.
- Mandatory 60fps animations.
