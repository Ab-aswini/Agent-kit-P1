# Global Architecture — Agent-Kit V3.1

- **Core Engine**: The Hybrid Arsenal (IDE Augmentation via MCP)
- **Identity**: Open Source .agent-os
- **Tech Stack**: Markdown, JSON, Python (Scripts), Node.js (MCP Server)
- **Orchestration**: Tiered Governance bounded by the `@Antigravity-Directive.md`
- **Validation**: Deterministic Python scripts (`checklist.py`, `security_chaos_test.py`) exposed over MCP
- **Style**: Industrial-Pro Aesthetics / High Authority

## Key Paradigm Shifts (V3.1)
1. **Zero-Execution Engine**: We do not run LLMs. We empower existing IDE AIs (Cursor, Antigravity) with our tools.
2. **Semantic Memory Sync**: AST-aware ground-truth extraction via `memory_sync.py` replaces static halluciantions.
