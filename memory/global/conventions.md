# Global Conventions

## Development Workflow
1.  **Socratic Gate**: Ask 3 questions for any task > S.
2.  **2-Phase Orchestration**: Separate Planning (PLAN.md) from Execution.
3.  **PRD First**: No backend/frontend work without a PRD in `docs/prd/`.

## Coding Standards
- **Naming**: camelCase for JS/TS, snake_case for Python.
- **Architecture**: Modular, service-oriented.
- **Security**: Zero Trust, Assume Breach.

## AI Orchestration
- **Agent IDs**: Always use ID in communication (e.g., SFS-001).
- **Context**: Load relevant `.agent-os/skills/` before complex tasks.
