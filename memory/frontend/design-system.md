# Design System (Agent-Kit V3.1)

- **Colors**: Strict adherence to the Purple Ban (#000000, #FFFFFF, Industrial Grays).
- **Typography**: Inter (Headers) & Fira Code (Technical/Monospace).
- **Components**: UI/UX Engine yields raw HTML/CSS structural components via MCP output. Deep focus on micro-interactions and non-safe layouts.
- **Vibe**: Elite, Autonomous, High-Authority.
